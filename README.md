# Proyecto Final — Data Analyst
## TripleTen Bootcamp | Telecomunicaciones & Análisis de Datos

## Descripción general
Proyecto final del bootcamp de Análisis de Datos compuesto por tres módulos
interrelacionados: análisis de operadores de telecomunicaciones, pruebas A/B
y consultas SQL aplicadas a un caso de negocio real.

---

## Módulo 1 — Telecomunicaciones: Identificación de operadores ineficaces
**Objetivo:** Identificar operadores con bajo desempeño mediante el análisis
de patrones operativos y métricas de rendimiento en una empresa de telecom.

**Proceso:**
- Limpieza y exploración de datos con Python / Pandas
- Conexión y consulta a base de datos con SQL
- Identificación de patrones de ineficiencia operativa
- Generación de insights y recomendaciones accionables

**Herramientas:** Python · Pandas · SQL · Power BI

---

## Módulo 2 — Test A/B: Sistema de recomendaciones
**Objetivo:** Evaluar el impacto de un sistema de recomendaciones mejorado
en el embudo de conversión de una tienda en línea internacional.

**Detalles del experimento:**
- Prueba: `recommender_system_test`
- Grupos: A (control) vs B (nuevo embudo de pago)
- Audiencia: 15% de nuevos usuarios región UE
- Periodo: diciembre 2020 — enero 2021
- Hipótesis: mejora mínima del 10% en cada etapa del embudo
  `product_page` → `product_card` → `purchase`

**Herramientas:** Python · Pandas · Estadística inferencial

---

## Módulo 3 — SQL: Análisis de base de datos de libros
**Objetivo:** Analizar una base de datos de una startup de libros para
generar una propuesta de valor para un nuevo producto, en el contexto
del cambio de hábitos de lectura durante la pandemia.

**Tablas analizadas:** `books` · `authors` · `publishers` · `ratings` · `reviews`

**Herramientas:** SQL · Python · Pandas
